# 🚀 AIGETAI — Scalable Frontend Experience System

AIGETAI is a modern, component-driven frontend platform engineered to deliver high-quality digital service experiences. It showcases capabilities across web development, mobile applications, AI systems, and custom software solutions.

The project is designed with a strong emphasis on **scalable architecture, reusable UI systems, and performance-driven interactions**, making it suitable as a production-ready frontend foundation.

---

## 🧩 Core Capabilities

- ⚡ Fully redesigned, modern **homepage experience**
- 🧱 Modular, reusable **component architecture**
- 🎯 Smooth UI interactions powered by **Framer Motion**
- 🖱️ Drag-enabled horizontal **service exploration (WhatWeDo)**
- ❓ Interactive **FAQ accordion with animated transitions**
- 🎡 Dynamic **carousel and showcase components**
- 🎨 Clean, maintainable **CSS structure**

---

* **Data-Driven UI** — Easy future API integration

## ⚠️ Asset Handling Strategy

* All images inside `src/assets` are imported via ES modules
* Avoid direct relative string paths inside data files
* Maintain consistent naming conventions (case-sensitive)
